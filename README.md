Front-End:

(i) HTML5
(ii) CSS3
(iii) JavaScript

Back-End:

(i) PHP
(ii) MySQL

Public User:

(i) Search Product
(ii) View Product
(iii) Create User Account

Signin User:

(i) Search Product
(ii) View Product
(iii) Create Order
(iv) Change Email & Password
(v) Can View Previous Order with UPDATE and DELETE

Admin:

(i) Add New Product
(ii) Update Product
(iii) Delete Product
(iv) Confirm Order
